# Frontend Module

This directory contains the Angular source code for the client-side application.

## Setup Instructions

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm start
   ```
   The application will be available at `http://localhost:4200`.

## Key Features
- **Authentication:** Login screens for Residents, Admin, and Security.
- **Dashboard:** Role-specific views.
- **WebSockets:** Real-time updates for visitors and parcels.
